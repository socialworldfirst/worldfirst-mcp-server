MCP Tools

10 tools your AI agent
can call right now

Every tool is designed with rich descriptions so LLMs understand when and how to use each capability. Install once, every MCP-compatible agent works.

💰

check_balance

Account

Query World Account balances across all currencies. Check available funds before making payments.

📊

get_fx_rate

FX

Get indicative FX rates for any currency pair. Informational — use for comparisons and estimates.

⚡

get_fx_quote

FX

Get a tradeable FX quote with live price and expiry. Ready to lock and execute.

🔒

lock_fx_rate

FX

Accept a quote and lock the rate. Protect against FX volatility on large supplier payments.

👥

list_beneficiaries

Supplier

List all saved suppliers and payees. Find beneficiary IDs for payment execution.

add_beneficiary

Supplier

Register a new supplier for future payments. Supports Chinese bank accounts and local bank codes.

💸

How It Works

Your AI agent pays a
Chinese supplier in 6 steps

No human clicks. No bank portal. Your agent handles the entire payment flow — from balance check to settlement confirmation.

1

Check balance

Agent confirms $12,450 USD available

2

Get FX quote

Live rate: 1 USD = 7.24 CNY, expires in 30s

3

Lock rate

Rate locked — protected from volatility

4

Verify supplier

Bank account matches business registration ✓

5

Execute payment

¥36,200 CNY sent to supplier's bank

6

Confirm settlement

Settled same-day via MYbank routing

// Your AI agent's conversation:

User: "Pay our Shenzhen supplier
       $5,000 worth of CNY"

// Agent calls WorldFirst MCP tools:

check_balance({ currency: "USD" })
// → $12,450.00 available

get_fx_quote({
  sell_currency: "USD",
  buy_currency: "CNY",
  sell_amount: "5000.00"
})
// → 1 USD = 7.24 CNY, quote Q-8f3a

lock_fx_rate({ quote_id: "Q-8f3a" })
// → Rate locked ✓

verify_supplier({ beneficiary_id: "B-456" })
// → Verified against biz registration ✓

create_payment({
  beneficiary_id: "B-456",
  amount: "36200.00",
  currency: "CNY",
  purpose: "Invoice #1234"
})
// → Payment P-789 initiated

check_payment_status({ payment_id: "P-789" })
// → Settled (same-day) ✓
